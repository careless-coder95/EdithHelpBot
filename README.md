## тЪб Features

<details>
<summary><b>ЁЯФТ Lock System</b></summary>
<br>

| Command | Description |
|:---|:---|
| `/lock <type>` | Lock a specific type |
| `/unlock <type>` | Unlock a specific type |
| `/locks` | Show all active locks |

| `/unlockall` | Unlock everything at once ЁЯФУ |

## ЁЯЪА Deployment

<details>
<summary><b>ЁЯЦея╕П VPS Deployment</b></summary>
<br>

### 1 тАФ Server Setup
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3 python3-pip git screen -y
```

### 2 тАФ Clone & Setup
```bash
git clone https://github.com/careless-coder95/EdithHelpBot.git
cd EdithHelpBot
pip3 install -r requirements.txt
```

### 3 тАФ Configure
```bash
nano .env
# Fill in your values тЖТ Ctrl+X тЖТ Y тЖТ Enter
```

### 4 тАФ Run in Background
```bash
screen -S edith
python3 main.py
# Detach: Ctrl+A тЖТ D
# Reattach: screen -r edith
```

### 5 тАФ Auto Restart (systemd)
```bash
sudo nano /etc/systemd/system/edith.service
```
```ini
[Unit]
Description=EdithHelpBot
After=network.target

[Service]
User=root
WorkingDirectory=/root/EdithHelpBot
ExecStart=/usr/bin/python3 main.py
Restart=always
RestartSec=5

[Install]
WantedBy=multi-user.target
```
```bash
sudo systemctl daemon-reload
sudo systemctl enable edith
sudo systemctl start edith
sudo systemctl status edith
```

</details>
